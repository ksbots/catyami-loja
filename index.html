Limpeza de Arquivos Temporários (CMD)

REM Limpa temporarios do Windows del /q /f /s %TEMP%\* del /q /f /s C:\Windows\Prefetch\* del /q /f /s C:\Windows\Temp\*

Limpar RAM (PowerShell)

# Limpar lista de trabalho da memoria [System.GC]::Collect() [GC]::WaitForPendingFinalizers() EmptyTrimmed = 2

Otimizar Rede (CMD)

REM Flush DNS e otimizacao TCP/IP ipconfig /flushdns netsh int ip reset netsh winsock reset netsh interface tcp set global autotuning=normal netsh interface tcp set global rss=enabled

Jogos - Desabilitar Game Bar/ DVR (Registry)

reg add "HKCU\System\GameConfigStore" /v GameDVR_Enabled /t REG_DWORD /d 0 /f reg add "HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\GameDVR" /v AppCaptureEnabled /t REG_DWORD /d 0 /f

Plano de Energia Ultra Performance
